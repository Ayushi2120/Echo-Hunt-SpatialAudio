# 🎧 Echo Hunt

> "Follow the echoes, uncover the unseen."

## 🌌 Overview

Echo Hunt is an immersive XR experience where players navigate virtual environments using spatial audio cues instead of traditional visual guidance. The project explores how sound can become the primary interaction mechanism, creating engaging and accessible gameplay within extended reality environments.

---

## ✨ Features

- Real-time 3D spatial audio
- Interactive XR environments
- Audio-guided exploration
- Hidden object mechanics
- Immersive UI/UX
- Accessibility-focused gameplay

---

## 🛠️ Tech Stack

- Unity
- C#
- XR Interaction Toolkit
- Spatial Audio SDK
- Blender

---

## 🎮 How It Works

Players enter a virtual environment and rely on sound intensity and direction to locate hidden objects and complete objectives. The experience encourages auditory awareness and demonstrates the potential of audio-first interaction in XR.

---

## 🚀 Future Enhancements

- Multiplayer mode
- AI-generated sound puzzles
- Voice interaction
- Additional maps and missions
- Cross-platform XR support
